## Installation

Next 설치

```
npm install
```


Next 실행

- development 환경
```
npm run dev -H {HOST_IP} -p {PORT}
```

- build 환경
```
npm run build
```
이후
```
npm run start -H {HOST_IP} -p {PORT}
```
